Architecture

Substrate

The underlying Substrate runtime provides deterministic execution primitives and a modular runtime environment.

Governed Lane

A governed lane is subject to explicit governance actions for upgrades and policy changes.

Ungoverned Lane

Ungoverned lanes operate under fixed contracts where deterministic behavior is required without active governance.

Telemetry Daemon

Collects runtime metrics, publishes observability signals, and feeds the metrics API and cockpit dashboards.

Orchestrator

Coordinates continuity markers, checkpoints, and corrective flows across replicas.

Metrics API

Serves runtime telemetry for dashboards and external observability systems.

Cockpit

The public-facing observability UI is available at secscockpit.live.

Continuity Markers

Markers provide checkpoints that define narrative closure and replay points for deterministic reconciliation.

Collapse Paths

Defined strategies for safely collapsing or migrating state when required by governance or operational constraints.
